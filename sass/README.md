## Naming Conventions<a name="NamingConventions"></a>

**DO** prefix an `_` to components (files that are meant to be imported).

```css
/* File profile.scss */

@import "_profile-calendar.scss";
```
<br>

**DO** use _spinal-case_ variables.

```css
$main-font: Arial;
```
